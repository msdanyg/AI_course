# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is an Obsidian vault for AI course notes and materials. It is not a traditional code repository.

## Vault Structure

- Notes are stored as Markdown files in the root and subdirectories
- `.obsidian/` contains Obsidian configuration and plugins
- `Tags/` folder for tag-based organization

## Installed Plugins

- **Templater** - Template system for creating structured notes
- **QuickAdd** - Quick capture and macro system
- **Make.md** - Enhanced file/folder views and spaces
- **Table Editor** - Improved table editing
- **Obsidian Git** - Version control integration
- **Style Settings** - Theme customization

## Working with This Vault

When assisting with this vault:
- Use the `mcp__obsidian-semantic__vault` tool for file operations
- Use `mcp__smart-connections__search_notes` for semantic search across notes
- Markdown files follow Obsidian conventions (wikilinks, frontmatter, callouts)
