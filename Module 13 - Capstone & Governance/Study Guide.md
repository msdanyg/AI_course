|--------|-----------|-------------|
| **Accountability** | "AI drafts. Humans send." | You own every output that bears your name |

## Verification Standards by Risk Level

| Output Type | Verification Standard |
|-------------|----------------------|
| Internal brainstorming | Quick review for relevance |
| Draft internal docs | Review for accuracy and tone |
| Customer-facing emails | Full review + fact-check key claims |
| Data analysis | Verify methodology + spot-check calculations |
| Public content | Multi-person review + PR/Legal if needed |
| Financial/legal docs | Expert review + compliance check |

---

## The Spot-Check Protocol

For data analysis, don't verify everything—use this protocol:

1. **Methodology**: Did the AI approach the problem correctly?
2. **Spot-check**: Manually verify 3-5 data points
3. **Logic**: Do conclusions follow from the data?
4. **Edge cases**: "What about [unusual scenario]?"

*If all pass: trust the output. If any fail: investigate.*

---

## The Transparency Prompt

Before using critical AI output, ask:

```
1. What assumptions did you make?
2. What information would change your answer?
3. What are you least confident about?
4. What should I verify independently?
```

## Measuring ROI

**Time-Quality Matrix:**
| | Lower Quality | Same/Higher Quality |
|---|---|---|
| **Faster** | Risky Acceleration | **The Sweet Spot** |
| **Same Speed** | Net Negative | Modest Win |

**Net Time Saved** = Task time without AI - (AI time + Review time + Fix time)

**Monthly Review Questions:**
1. What % of work involved AI?
2. Where did I save the most time?
3. Where did AI most improve quality?
4. Where did it add friction?
5. What should I learn next?

## Common Governance Mistakes

| Mistake | Problem | Solution |
|---------|---------|----------|
| Same review for everything | Wasted time or missed risk | Scale with consequence |
| "AI wrote it" excuse | No accountability | Own every output |
| No Red Team check | Blind spots persist | Always critique before finalizing |
| Skipping verification | Errors reach audience | Follow the protocol |
