# Bonus Module: SARA
## Lab Exercise: Meet Your Command Center

**Duration:** 15-20 minutes
**Tools Required:** Claude.ai with access to the SARA Project
**Prerequisites:** Module 4 (Personal Projects & Folders)

---

## Objective

Open SARA, test its capabilities for your specific role, practice feeding it external context and discover its boundaries firsthand.

---

## Step 1: Open SARA (2 minutes)

Navigate to [claude.ai](https://claude.ai). In the left sidebar, find the **SARA** Project (it may appear as "SARA: Sales Assistant Response Agent" or similar).

If you don't see it, check whether the Project has been shared with you. Ask your manager or Daniel Glickman for access.

**Star the Project** so it stays pinned in your sidebar for quick access.

Start a **new chat** inside the SARA Project.

---

## Step 2: Orientation Prompt (3 minutes)

Type this into SARA:

```
How can you help me?
```

If you haven't set personal preferences in Claude, add your role:

```
I'm a [your role: AE / SDR / CSM / SE]. How can you help me?
```

Read SARA's response carefully. Note:
- [ ] Does it describe competitive positioning support?
- [ ] Does it mention specific file types it has access to (battlecards, case studies, messaging docs)?
- [ ] Does it tailor recommendations to your role?

---

## Step 3: Role-Based Scenario (5 minutes)

Based on your role, paste one of these prompts into SARA:

**If you're an AE or Sales Manager:**
```
I'm on a call with a prospect who currently uses Teramind. What are the top 3 positioning angles I should emphasize and what discovery questions should I ask?
```

**If you're an SDR:**
```
I just cold-called a prospect who said they implemented Workday last quarter and aren't looking at new vendors. How should I position ActivTrak as complementary?
```

**If you're a CSM:**
```
A customer is asking why they should renew instead of switching to Hubstaff. Give me a renewal positioning brief with key value points.
```

**If you're an SE:**
```
I need to demo ActivTrak against Insightful next week. What are our strongest technical differentiators and what should I emphasize?
```

**If you're in another role:**
```
I need to write an internal summary of why ActivTrak beats Teramind for a VP who asked. Keep it to 3 paragraphs with our positioning language.
```

Evaluate the response:
- [ ] Does it use ActivTrak's approved positioning language?
- [ ] Does it reflect the "Insights, Not Oversight" philosophy?
- [ ] Is it specific enough to use in a real conversation?

---

## Step 4: Feed External Context (5 minutes)

Find a real piece of text from your work. This could be:
- A snippet from a Slack conversation about a customer
- A paragraph from a support center article
- Notes from a recent meeting (Granola or otherwise)
- A few bullet points you jotted down about an account

Paste it into SARA with this framing:

```
Reframe the following through ActivTrak's competitive positioning. I need to use this in a [email / call / presentation] with a [prospect / customer / internal stakeholder]:

[Paste your text here]
```

Evaluate the response:
- [ ] Did SARA maintain the core facts from your source material?
- [ ] Did it align the language with ActivTrak positioning?
- [ ] Would you feel confident sending this (with your edits) to the intended audience?

---

## Step 5: Boundary Test (3 minutes)

Ask SARA something it should NOT be able to do well:

```
Build me a complete QBR deck for my customer including their specific usage data and ROI calculations.
```

Observe how SARA responds:
- [ ] Does it attempt to fabricate a full deck, or does it redirect you?
- [ ] Does it explain what it can and can't help with?
- [ ] Does it suggest where to go for what it can't do (e.g., the POV Project, support center)?

---

## Reflection

```
1. How does SARA's output compare to what you'd produce on
   your own in the same timeframe?
   [ Much faster and better / Faster but needs editing / About the same ]

2. What other context could you feed SARA regularly to make
   it more useful for your specific accounts?


3. What's one workflow you do weekly that SARA could help with
   starting tomorrow?

```

## Success Criteria

- [ ] Opened SARA and received an orientation response
- [ ] Ran a role-based scenario and received positioning-aligned output
- [ ] Fed external context and received reframed output
- [ ] Tested a boundary and understood SARA's limitations
- [ ] Identified at least one weekly workflow where SARA adds value
