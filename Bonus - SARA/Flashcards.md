# Bonus Module: SARA
## Flashcards

---

### Card 1: What is SARA?

**Front:** What is SARA and what type of tool is it?

**Back:** SARA (Sales Assistant Response Agent) is a Claude Project pre-configured for ActivTrak's revenue teams. It contains custom instructions for positioning, plus uploaded battlecards, case studies and messaging documents. Squadron Metaphor: A "Mission-Specific Command Center" that's already built and operational.

---

### Card 2: Five Core Capabilities

**Front:** What are the five core things SARA can help with?

**Back:** (1) Live call support with competitive intel, (2) Objection handling with complementary positioning, (3) Outbound messaging creation with ActivTrak-aligned language, (4) Discovery question generation for competitive scenarios, (5) Context feeding — reframing raw notes through ActivTrak positioning.

---

### Card 3: SARA's Limitations

**Front:** What are SARA's limitations?

**Back:** SARA is NOT a product documentation tool (use the support center). It is NOT a QBR builder. It cannot replace human judgment — "AI drafts, humans send." Its intelligence files are manually maintained, so very new features may not be reflected yet.

---

### Card 4: Context Feeding

**Front:** What is "context feeding" and why does it make SARA more effective?

**Back:** Context feeding means copying text from external sources (Granola transcripts, Slack threads, support center articles, email threads) and pasting it into SARA with instructions to reframe through ActivTrak positioning. This gives SARA specific situational awareness beyond its static files. Squadron Metaphor: "Field Data Feeds" flowing into the Command Center.

---

### Card 5: Role-Based Usage

**Front:** How do different revenue roles use SARA differently?

**Back:** AEs use SARA for mid-call competitive intel and proposal language. SDRs use it for cold call objection handling and outbound sequences. CSMs use it for renewal positioning and case study lookup. SEs use it for technical differentiation and demo talking points. Same Command Center, different mission profiles.
