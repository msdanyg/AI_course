Module 1: The Cognitive Shift

Understanding the Reasoning Engine

Key Concept: Probabilistic Reasoning Engine

LLMs are not search engines. They're reasoning engines that predict the most probable next token based on patterns in training data.

The Water Glass Effect: Every piece of context you add to the conversation shapes the output. More relevant context = better results.

Context Window Management

  • Front-load important context
  • Remove irrelevant information
  • Structure with clear sections
  • Use XML tags for organization

Module 4: Agentic Data Analysis

The "No Math" Rule

The Calculator Problem

LLMs predict tokens, they don't calculate. For data analysis:

  • Use code execution (Claude's Analysis tool)
  • Let Python/pandas do the math
  • Verify outputs with spot checks
Golden Rule: "Never trust an LLM to do math. Make it write code instead."

Module 6: Decision Hygiene

Beating Sycophancy

The Sycophancy Problem

LLMs tend to agree with you. Combat this with:

  • Neutral framing - Don't telegraph your preference
  • Red teaming - Ask for counterarguments
  • Pre-mortems - "What could go wrong?"
Anti-Sycophancy Prompt: "Challenge this idea. What are the strongest arguments against it?"
