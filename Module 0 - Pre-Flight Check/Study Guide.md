## The Squadron Framework

Different employees have different tools. Think in **roles**, not specific tools.

| Squadron Role | Purpose | Example Tools |
|---------------|---------|---------------|
| **Mission Control** | Reasoning, writing, coding | Claude, ChatGPT, Claude Code, Cursor |
| **Recon & Radar** | Research, grounding | Gemini, Bing AI, NotebookLM |
| **Flight Recorder** | Meeting capture | Granola, Chorus.ai, Zoom AI |
| **Specialized Units** | Domain-specific | Figma AI, Zendesk AI, ChurnZero |

**Key insight:** Mission Control works with what you give it. Recon & Radar fetches what's out there. Flight Recorder captures what happens.

---

## Traffic Light Protocol

| Zone | Data Types | Action |
|------|------------|--------|
| **Green** | Customer data, internal docs, analytics, business communications | Use freely. Review outputs before external sharing. |
| **Yellow** | Marketing with PII, competitive intel with sources, NDA data | Can analyze, but outputs may need PR/Legal review. |
| **Red** | API keys, passwords, credentials, security configs | Never share with any AI tool. |

---

## Core Principle

> **"AI drafts. Humans send."**

- The risk is in **unreviewed output**, not in analysis
- You clear missions for takeoff
- You own what leaves the runway
- Always review before external distribution

---

## Decision Framework

Before pasting data into AI, ask:

1. **What is this data?** (Customer info? Credential? Internal doc?)
2. **What zone?** (Green/Yellow/Red)
3. **What will I do with output?** (Internal? External?)
4. **Have I reviewed before sending?** (Never auto-send)

---

## Solo Pilot vs. Squadron Leader

| Aspect | Solo Pilot | Squadron Leader |
|--------|------------|-----------------|
| Context | Minimal | Rich (background, goals, constraints) |
| Structure | Freeform questions | Organized requests, clear outputs |
| Continuity | New chat every time | Projects and persistent knowledge |
| Review | Copy-paste directly | Verify, refine, then use |

## Common Mistakes

| Mistake | Problem | Solution |
|---------|---------|----------|
| Sharing credentials with AI | Security risk | Red zone data—never share |
| Auto-sending AI outputs | No human review | Always review before external send |
| Using one tool for everything | Suboptimal results | Match tool to task |
| Minimal context in prompts | Generic outputs | Provide background, goals, constraints |
